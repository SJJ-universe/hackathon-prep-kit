# 분야별 고득점 빌드카드 — 당일 시제가 뭐가 나오든 펴서 바로 착수

> 용도: 대회 당일 시제(미션 키워드)가 공개되면, 해당 분야 카드를 펴서 30분 안에 "무엇을 만들지·MVP 1~2기능·핵심 공개데이터(ID)·차별점·당일 함정"을 확정하고 빈 새 저장소에서 곧장 코딩 착수하기 위한 한 장짜리 작전판.
>
> 반입 합법선: 이 문서는 지식·아이디어·공개데이터 탐색 결과·프롬프트 요령일 뿐 완성코드·주제맞춤 코드가 아니다. 당일 코드는 빈 새 저장소에 처음부터 작성한다.
>
> 정직성 경고(필독): 아래 분야 점수(44~70)·"한국 공백"·"미존재" 판정은 전부 보수 추정이며 best-effort 조사 결과다. "한국 최초/유일" 같은 대외 주장 전에는 반드시 개별 재검증한다. 데이터셋 ID·자동승인 여부는 `04_data/catalog.md` 실검증 근거를 옮긴 것이나, data.go.kr 응답 표는 Swagger 동적렌더라 정적 조회로 안 읽히는 경우가 많다 → 표시한 ID·필드는 [당일 확인](활용신청 후 응답 1건이면 확정)을 원칙으로 한다. URL·숫자 지어내기 금지, 미확인은 "미검증"으로 표기.

## B. 추천 아이디어 짧은 카드 (무엇을·왜 고득점·시연 1장면·필요 데이터/키·당일 함정)

### S1. 받을자격:실업 (통과 1위, 본진 확정 후보)
- 무엇을: 퇴사 사유를 평소 말투 한 줄로 입력→고용보험법 시행규칙 [별표2] 근로자 정당한 이직사유 중 어디에 가까운지 매핑+지금 모을 증거+이직 다음날+12개월 시효 D-day.
- 왜 고득점: 외부 라이브 API 0(법령 텍스트 룰표)이라 4h 완주가 가장 확실. "자격 판정"이 아니라 "행동 번역"이라 고용24와 비중복. 자유발화→별표2 의미매핑이 규칙 불가=AI load-bearing.
- 시연 1장면: "심사위원님 퇴사 사연 한 줄 불러주세요"→즉석 타이핑→Claude가 별표2 실텍스트 위에 라이브 매핑+원문 verbatim 인용+신뢰도. 못 찾으면 "포기 말고 고용센터" 캐치-올 폴백.
- 필요 데이터/키: 별표2(law.go.kr `bylNo=0002`, 키불필요·공개텍스트) / 제40조②(유급 180일)·제48조①(12개월 시효) / 고용복지+센터 `15066368`(20250901판) / 1350.
- 당일 함정: 근로자 별표2 헤더 `<개정 2024.7.1>` 육안 재확인(예술인 별표2의2=2024.12.31과 혼동 금지). 호 수는 1~13+3의2(직장 내 괴롭힘)=실제 14개지만 사용자 화면엔 숫자 권위값 미제시. 진단 대상은 근로자 한정, 예술인/노무제공자는 별표2의2/2의3 안내 분기. 자격 단정 금지=출력 화이트리스트.

### S2. 한걸음 (통과 2위, 유일하게 엔진이 다른 백업)
- 무엇을: 스토킹·교제폭력 피해 서술→AI가 본인 문장을 6하원칙 시간순으로 제안 정렬(이용자 확정)+위험신호 설명+무전송 메모 PDF. AI는 진술서를 "작성"하지 않는다.
- 왜 고득점: 손에 잡히는 PDF 산출물=시연 한 방. 취약층 공공성 높음. 거짓 안심("안전") 출력 불가=출력 화이트리스트. 무전송·무로그인.
